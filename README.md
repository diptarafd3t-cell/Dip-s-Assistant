# Hey D - Android AI Assistant
A voice-activated AI assistant powered by **Gemini 1.5 Flash**.

## Features
- Voice Wake Word: "Hey D"
- AI Brain: Google Gemini
- Platform: Android (Pydroid 3 / Termux)

## Setup
1. Clone this repo.
2. Install requirements: `pip install -r requirements.txt`
3. Add your API Key to `config.py`.
4. Run `main.py`.
